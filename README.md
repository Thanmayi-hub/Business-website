**Responsive Business Website**

**Project Name**: Responsive Business Website

**Objective**: This project is developed using HTML, CSS, and JavaScript. It is a responsive website that works on desktop, tablet, and mobile devices.
Pages
Home
About
Services
Gallery
Contact
HTML

I used HTML to create the structure of the website.
**I used semantic tags like:**
<header>
<nav>
<section>
<footer>
CSS

**I used CSS to:**
Add colors
Design the navigation bar
Create service cards
Style buttons
Add hover effects
Make the website responsive using Media Queries
JavaScript

I used JavaScript to validate the contact form.
**It checks whether:**
Name is entered
Email is entered
Message is entered

**If any field is empty, it shows:**

"Please fill all fields."

**Otherwise, it shows:**

"Message Sent Successfully!"

**Responsive Design**

I used Media Queries so the website adjusts automatically for mobile, tablet, and desktop screens.

**Conclusion**

This project is a simple responsive business website with multiple pages, attractive design, navigation, and a contact form with JavaScript validation.
